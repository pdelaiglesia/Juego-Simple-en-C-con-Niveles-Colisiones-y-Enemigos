# Juego en C++ con Niveles, Colisiones y Enemigos

Este proyecto implementa un videojuego en C++ ejecutado en consola. Incluye colisiones, enemigos móviles, proyectiles, paredes, botones interactivos y un sistema de vidas. Cada nivel introduce nuevos obstáculos y mecánicas, manteniendo una progresión de dificultad.

## Características principales
- Movimiento del jugador con detección de teclas en tiempo real.
- Sistema de colisiones con paredes, enemigos y elementos interactivos.
- Enemigos con movimiento autónomo.
- Proyectiles y detección de impacto.
- Botones y zonas que activan cambios en el entorno.
- Gestión de vidas y avance de niveles.
- Diseño modular para facilitar ampliación del juego.

## Tecnologías utilizadas
- C++
- Funciones de consola para renderizado básico
- Manejo de entrada en tiempo real (sin necesidad de presionar Enter)

## Estructura básica del código
- `player`: lógica de movimiento y colisiones.
- `enemy`: comportamiento de enemigos.
- `projectile`: gestión de disparos.
- `level`: carga y configuración de mapas.
- `main.cpp`: bucle principal del juego.

## Cómo ejecutar
1. Compilar con cualquier compilador compatible con C++11 o superior.
2. Ejecutar el archivo generado desde la consola.
3. Jugar utilizando las teclas de movimiento definidas en pantalla.
