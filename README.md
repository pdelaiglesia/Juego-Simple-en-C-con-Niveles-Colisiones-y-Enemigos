# Juego-Simple-en-C-con-Niveles-Colisiones-y-Enemigos
Juego en C++ para consola con niveles progresivos, detección de colisiones, enemigos móviles, proyectiles, paredes, botones y sistema de vidas. Incluye detección de teclas en tiempo real, diseño por capas y lógica modular para gestionar movimientos y escenarios.
